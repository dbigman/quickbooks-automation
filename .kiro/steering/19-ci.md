# CI & Quality

**Pipeline (push/PR)**
1) `ruff check .`
2) `black --check .`
3) `isort --check-only .`
4) `mypy src`
5) `pytest`

**Policies**
- Fail on any violations.
- Keep CI under ~5 minutes; parallelize tests if needed.
- Pin exact tool versions in `pyproject.toml`.
